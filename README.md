# Link in Bio

Platform modern untuk mengelola dan membagikan tautan penting Anda dalam satu halaman.

## Fitur

- **Multiple Links**: Tambah dan kelola banyak tautan dalam satu halaman
- **Customizable Design**: Personalisasi tampilan dengan mudah

## Setup

1. **Install dependencies**:
   ```bash
   pnpm install
   ```

2. **Jalankan development server**:
   ```bash
   pnpm run dev
   ```

3. **Buka browser**:
   ```
   http://localhost:3000
   ```

## Tech Stack

- **Next.js 15** - React framework dengan App Router
- **Tailwind CSS** - Utility-first CSS framework
- **Prisma** - ORM untuk database management
